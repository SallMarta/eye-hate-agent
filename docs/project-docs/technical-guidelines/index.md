This document is the index registry for the durable technical guideline documents of the Eye Hate Agent (EHA) repository.

## 7. Active Guidelines

As a result, standard JavaScript styling standards (e.g., Prettier/ESLint defaults), standard Node.js practices, and EHA compact rules are sufficient, and no extra guideline documentation is active.
